Project Proposal

의약품 포장 이미지 기반 복약 안전 정보 검색을 위한 OCR-RAG와 Image-KG Retrieval 비교

약 포장 이미지에서 OCR 텍스트 기반 검색과 이미지-지식그래프 기반 검색의 정확도를 비교하고, 두 신호를 결합한 Hybrid GraphRAG 기반 복약 안전 정보 검색 시스템을 제안한다.

주 실험: 포장 이미지 비교: OCR-RAG vs Image-KG vs Hybrid 출력: 복약 안전 카드

1. 프로젝트 개요

본 프로젝트는 사용자가 의약품 포장 이미지를 입력했을 때 해당 약품의 품목명, 성분, DUR 경고, 음식 상호작용, 복약 주의사항을 정확히 검색하는 AI 시스템을 구현하는 것을 목표로 한다.

기존 의약품 복용 도우미 앱 기획은 사진, 음성, 텍스트 입력을 모두 활용하는 멀티모달 서비스였으나, 본 프로젝트에서는 핵심 문제를 약 포장 이미지 기반 의약품 정보 검색으로 좁힌다. 동일한 약 포장 이미지에 대해 OCR 기반 검색과 이미지 기반 KG 검색의 성능을 비교하고, 두 방식을 결합한 Hybrid GraphRAG가 더 안정적인 검색 성능을 보이는지 검증한다.

2. 문제 정의 및 연구 질문

일반 사용자는 약 포장 이미지만 보고 정확한 품목명, 성분, 병용금기, 음식 상호작용 정보를 직접 확인하기 어렵다. 포털 검색이나 LLM 단독 질의는 출처가 불분명하거나 환각 가능성이 있으며, OCR 기반 검색은 이미지 흐림, 빛 반사, 부분 촬영, 포장 훼손, 글자 인식 오류에 취약하다.

주 연구 질문

약 포장 이미지 기반 의약품 정보 검색에서 OCR-RAG와 Image-KG Retrieval은 정확도와 안정성 측면에서 어떤 차이를 보이는가?

추가 연구 질문

OCR 결과와 이미지 기반 검색 결과를 결합한 Hybrid GraphRAG는 단일 방식보다 더 높은 약품 식별 정확도와 복약 안전 정보 검색 성능을 보일 수 있는가?

3. 프로젝트 목표

목표 내용
OCR-RAG 구현 약 포장 이미지에서 OCR 텍스트를 추출하고 이를 기반으로 의약품 정보를 검색한다.
Image-KG Retrieval 구현 약 포장 이미지를 product_image 노드로 KG에 연결하고 이미지 임베딩 유사도와 KG 관계 탐색으로 약품 정보를 검색한다.
Hybrid GraphRAG 구현 및 평가 OCR 점수와 이미지 점수를 결합하여 세 방식의 성능을 정량적으로 비교한다.

범위 조정

본 프로젝트의 주 실험은 포장 이미지 기반 검색으로 제한한다. 경구약제 이미지는 실제 앱 확장 가능성을 위해 보조 데이터로 남기되, 본 실험의 핵심 비교 대상에는 포함하지 않는다.

5. 제안 방법

방법 설명
OCR-RAG 포장 이미지에서 OCR 텍스트를 추출한 뒤 약품명 후보를 기반으로 의약품 정보를 검색한다.
Package Image-KG Retrieval 포장 이미지를 임베딩으로 변환하고 KG의 product_image 노드와 유사도를 비교해 연결된 drug_item을 탐색한다.
Hybrid GraphRAG OCR 텍스트 유사도와 이미지 유사도를 결합하여 최종 약품 후보를 ranking한다. 
final_score = alpha * OCR_text_similarity + (1 - alpha) * image_similarity

초기 실험에서는 alpha = 0.5의 고정 가중치를 사용하고, 추가 실험에서는 OCR confidence에 따라 가중치를 동적으로 조정한다.

8. 실험 설계

실험 데이터는 약 포장 이미지 30~100장 규모로 구성한다. 각 이미지에는 정답 품목기준코드, 품목명, 제조사, 성분명, DUR 규칙, 음식 상호작용, 근거 출처 라벨을 부여한다.

난이도 조건 목적
Easy 제품명이 선명한 정면 포장 기본 검색 성능 확인
OCR-hard 흐림, 빛 반사, 기울어짐, 부분 촬영 OCR 실패 상황 검증
Image-hard 비슷한 포장 색상, 제조사, 디자인 이미지 retrieval 취약점 검증
Mixed-hard OCR과 이미지 검색 모두 어려운 사례 Hybrid 안정성 검증

9. 평가 지표

Top-1 Accuracy 1순위 후보가 정답 약품인지 평가한다.
Recall@K 정답 약품이 Top-K 후보 안에 포함되는지 평가한다.
MRR 정답 후보가 얼마나 상위에 배치되는지 평가한다.
Ingredient Match 정답 성분을 정확히 검색했는지 평가한다.
Rule Match DUR 및 음식 경고를 정확히 검색했는지 평가한다.
Evidence Recall@K 정답 근거 출처가 검색 결과에 포함되는지 평가한다.

12. 요약

본 프로젝트는 약 포장 이미지 기반 복약 안전 정보 검색에서 OCR-RAG와 Image-KG Retrieval의 성능을 비교하는 것을 목표로 한다. OCR-RAG는 이미지에서 추출한 텍스트를 기반으로 의약품 정보를 검색하는 방식이며, Image-KG Retrieval은 약 포장 이미지를 product_image 노드로 지식그래프에 연결하고 이미지 임베딩 유사도 및 KG 관계 탐색을 통해 약품, 성분, DUR, 음식 상호작용 정보를 검색하는 방식이다.

최종적으로는 OCR 텍스트 유사도와 이미지 유사도를 결합한 Hybrid GraphRAG를 구현하여, 단일 방식 대비 약품 식별 정확도와 복약 안전 정보 검색 성능이 개선되는지 정량적으로 평가한다.

본 기획서는 연구 및 교육 목적의 프로토타입 설계를 기준으로 하며, 실제 복약 판단은 전문가 상담과 공식 의약품 정보를 우선해야 한다.
